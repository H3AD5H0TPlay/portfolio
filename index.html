Szolgáltatások

Frontend fejlesztés

Frontend fejlesztés közben törekszem a modern és letisztult dizájn elkészítésére, reszponzitivással és animációkkal, effektekkel együtt, React használatával.

Backend fejlesztés

Backend fejlesztés közben Spring Boot vagy Django keretrendszereket használok, melyek modernek és a maguk módján egyszerűek, mégis biztonságosak és gyorsak.

Fullstack fejlesztés

Az előző két pontban feltűntetett tudásomat egybesűrítve egy teljes projektet is képes vagyok elkészíteni, lásd például a Github oldalamon. Spring Boot keretrendszer esetén Thymeleaf framework, míg Django keretrendszer esetén React framework használatát alkalmazom.

Szoftverfejlesztés

Programozási nyelvekben szerzett jelentős tapasztalataimnak köszönhetően képes vagyok szoftvereket, alkalmazásokat készíteni akár telefonra vagy egyéb eszközre is.
